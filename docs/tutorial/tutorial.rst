=====================
Tutorial: A Basic Workflow
=====================

.. _slides: https://slides.com/johanneskoester/snakemake-tutorial

In this tutorial, we will design and execute our first workflow with Snakemake, going over the basic concepts and building blocks of the system.

We will try to solve a common task in Bioinformatics: Read Mapping.
However, Snakemake is a general-purpose workflow management system for any discipline.
We have tried to ensure that no bioinformatics knowledge is needed to understand the tutorial.

There are also slides_ available for this tutorial.
